Projects

A console-based game where the computer generates a random number, and the user attempts to guess it with hints indicating whether the guess is too high or too low.

Task 3 – Tic-Tac-Toe Game

A two-player console-based Tic-Tac-Toe game that allows players to take turns, validates moves, detects winning conditions, and identifies draw situations.
